## 📝 Descripción
Aplicación interactiva desarrollada para el curso de **Talento Tech - Nivel Integrador**. Este proyecto analiza la relación entre variables climáticas y el consumo energético global utilizando un dataset de Kaggle.

## 📊 Características
- Landing page explicativa.
- Análisis de tendencias temporales.
- Correlaciones entre temperatura y consumo.
- Distribución de fuentes de energía.
